# inVision U
AI анализ мотивационных писем для HR комиссии.
## Технологии
Backend: Go и стандартная библиотека net/http
Frontend: HTML/CSS/JS
## Установка и запуск
### Требования
- Go 1.21 или выше
### Запуск
1. Склонируйте репозиторий
2. Откройте терминал в папке проекта
3. Выполните команду:
```bash
go run main.go
